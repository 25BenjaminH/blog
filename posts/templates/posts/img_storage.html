Geographical Data Visualization for High Dimensional Symbolic Data

Due to the advent of data science, generating and collecting data became much easier, causing the scale of data to rapidly grow, and its data types have become more diverse.

A newly developed research is focusing on the computation efficiency and statistical analysis methods for high-dimensional geographical data. The current focus is on the numerical value of a single variable corresponding to its data value through the color level, and the geographic information is presented in the color of its corresponding color level. However, drawing the content of each variable on the map is required for presenting high-dimensional geographic data, causing this method to be inefficient and difficult for the researchers to use. Besides, if the data is symbolic and contains geographic information, the visualization of the map is even more challenging to achieve.

This project proposes a new method for map visualization of high-dimensional big data. In this project, a data record of 214 countriesâ€™ Covid-19 cases within the 315 days in the year 2020 was used. Each day, the cases of the 214 countries were sorted using the ranking method because the average of the cases can be greatly affected by the outliers. By calculating the Euclidean distance of the ranked values, we can then draw SVD using two eigenvectors to show the relationships between the countries. With the drawn graph, we can then get the radian measures of the values. The HSV value for each country is lastly determined by using the radian measures on a HSV graph. The method proposed in this project will resolve the prior mentioned problems and provide big high dimensional geographical data efficiency.
